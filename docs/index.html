∑ Texmail

Turn LaTeX into email-ready images with one keystroke.

Install with one command:
curl -fsSL https://raw.githubusercontent.com/Franklinwang72/texmail/main/install.sh | bash

View on GitHub

Features

⌨️ Global Shortcut

Select text in any app, press ⌘⇧L. Formulas are rendered and replaced in-place.

📧 Email-Ready

Works with Gmail, Apple Mail, Outlook, and Thunderbird. HTML + RTFD output.

🔤 All LaTeX Delimiters

Supports $...$ , \(...\) , $$...$$ , and \[...\] — all standard LaTeX math modes.

🧠 Smart Rendering

Uses matplotlib for simple math. Auto-falls back to xelatex for amsmath, tikz-cd, CJK.

🎨 Computer Modern Font

The classic LaTeX look — same font used in academic papers worldwide.

⚡ Menu Bar App

Runs silently in background. No Dock clutter. Always ready when you need it.
